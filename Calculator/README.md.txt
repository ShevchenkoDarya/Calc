# Calculator with Plugins

Консольный калькулятор с динамической загрузкой плагинов на C++.

## Функции
- Базовые операции: +, -, *, /, ^
- Скобки: ()
- Плагины: sin(), ln() и другие (добавляются через DLL)

## Сборка
1. Откройте `Calculator.sln` в Visual Studio
2. Постройте решение (Ctrl+Shift+B)
3. Скопируйте .dll из папок плагинов в ./plugins

## Структура
- `/Calculator` - основное приложение
- `/PluginSin`, `/PluginLn` - плагины
- `/plugins` - папка для скомпилированных DLL

## Требования
- Visual Studio 2019 или новее
- C++17
- Windows
